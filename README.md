## Getting Started


### Installation

2. Clone the repo
   ```sh
   https://github.com/Rahugg/CRM-system-go-microservices.git
   ```
3. Install go packages
   ```sh
   go mod tidy
   ```
4. Launch docker-compose 
   ```
   docker-compose up
   ```
